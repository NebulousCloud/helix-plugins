Local Event

by Ice Bear#2034

  • Description
  • Commands
  • Configs
  • License

Adds in a new /LocalEvent command only heard from a certain distance, it appears in yellow rather than orange.

Players can use these commands to interact with this plugin

/LocalEvent

Make something perform an action that only those in a certain distance can see.

Servers can customize usage of this plugin with these settings

Local Event Distance (default: 560)

The maximum distance people can hear a local event, default is the standard yelling range.
