Player Flags

by SleepyMode

  • Description
  • Commands
  • License

Allows flags to be assigned to players rather than characters.

Players can use these commands to interact with this plugin

/PlyClearFlags

Removes all flags from a player.

/PlyGiveAllFlags

Gives a player all available flags.

/PlyGiveFlag

No description available.

/PlyTakeFlag

No description available.
